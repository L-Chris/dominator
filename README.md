# Dominator

> 面向知乎阅读场景的浏览器扩展，用于识别用户类型、辅助判断账号风险，并按配置折叠指定类型用户的回答。

![Dominator promotional banner](store-assets/edge/large-promo-1400x560.png)

## 项目简介

Dominator 是一个 Chrome / Edge 浏览器扩展。它会在知乎问题页中展示用户类型标签，帮助你更快识别回答来源与账号行为特征。

你也可以在扩展弹窗中选择需要过滤的用户类型。命中配置的回答会被折叠，页面上会保留一个展开按钮，需要时仍然可以查看原回答。

Dominator 不是裁决工具，也不会替你删除内容。它只是把可疑特征以更容易阅读的方式呈现出来，最终判断仍应由你结合上下文完成。

## 命名寓意

**Dominator** 取自《心理测量者》（PSYCHO-PASS）中的同名装置。原作里的 Dominator 会读取目标的心理状态并给出风险判断；本项目借用这个意象，用来表达“对公开内容进行多维度风险观察”的产品定位。

这里的 Dominator 不是裁决工具，而是一个辅助观察工具。它只基于公开内容生成用户类型和风险提示，帮助用户更快发现异常模式。

## 关于开发

本项目由 **Qwen3.7-Max** 深度辅助开发。从产品迭代、交互调整、架构设计到代码实现，主要开发流程都由 AI 协作完成。

## 主要功能

- 在回答作者旁显示用户类型标签。
- 在头像悬浮弹窗中显示更多辅助标签。
- 支持配置需要折叠的用户类型。
- 被折叠的回答可以随时手动展开。

## 可识别的用户类型

Dominator 会把用户归类为以下类型之一：

- 普通用户
- 垂类爱好者
- 品牌粉
- 软广号
- 硬广导流号
- 模板搬运号
- 舆论带节奏号
- 协同水军号
- 机器人式账号

## 安装使用

发布到扩展商店后，可以直接从 Edge Add-ons 安装。

如果你需要本地安装测试：

1. 下载或构建扩展包。
2. 打开 Edge / Chrome 的扩展管理页面。
3. 开启开发人员模式。
4. 选择“加载解压缩的扩展”。
5. 选择构建后的扩展目录。
6. 打开知乎问题页，等待作者名旁出现用户类型标签。
7. 点击浏览器工具栏里的 Dominator 图标，配置需要折叠的用户类型。

## 隐私说明

Dominator 会读取当前知乎页面中的公开用户信息和回答样本，用于生成用户类型与风险分析结果。

扩展不会读取知乎密码、私信、支付信息，也不会替用户发布、点赞、评论或关注。

你的过滤配置会保存在浏览器扩展存储中，用于在页面中折叠指定类型的回答。
